Script Reference

ClipChannel

Base type

Base

Description

A ClipChannel object. The ClipChannel member of the Base type provides access to the clip channel. Use System.GetClipChannel(number as Integer), whereas number >= 0, to get a ClipChannel instance. See also examples section ClipChannel.
