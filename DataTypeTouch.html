Script Reference

Touch

Properties

Direction As Vertex [read-only]Returns the touch's pointing direction (e.g. direction of finger) as an uniform vector.
Event As Integer [read-only]Returns the touch event type (TOUCH_DOWN, TOUCH_MOVE, TOUCH_UP).
Index As Integer [read-only]Returns the index of the touch object.
Pressure As Double [read-only]Returns the normalized pressure value in the range 0.0 to 1.0.
Timestamp As Double [read-only]Returns the timestamp of this touch.
Trace As Trace [read-only]Returns the parent trace object.
Valid As Boolean [read-only]Returns true if the object is valid.
xy As Vertex [read-only]Returns the touch position in screen-coordinates.
