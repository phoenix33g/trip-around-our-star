Script Reference

String

Description

A character string. Use the '&' operator to concatenate strings.

Properties

Length As Integer [read-only]Gets the length of the string.

Checking the length of a doublebyte string:
You have to make sure to use the correct function for calculating double byte strings. Normal Doublebyte Fonts (Asian/Cyrillic) are calculated wrong if you use the string.length() function.
Solution: To get the prober length, you have to convert it using Utf8ToAnsi()
